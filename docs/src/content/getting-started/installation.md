# Installation

## Via npm

```bash
npm install @better-pwa/core
```

## Via pnpm

```bash
pnpm add @better-pwa/core
```

## Via yarn

```bash
yarn add @better-pwa/core
```

## Verify Installation

```bash
node -e "
const { createPwa } = require('@better-pwa/core');
console.log('✅ better-pwa installed successfully');
console.log('Version:', require('@better-pwa/core/package.json').version);
"
```

## Optional: Framework Adapters

```bash
# React
npm install @better-pwa/adapter-react

# Vue
npm install @better-pwa/adapter-vue

# Svelte
npm install @better-pwa/adapter-svelte
```

## Project Structure

After installation, your project should look like:

```
my-app/
├── src/
│   ├── main.ts          # Entry point
│   ├── app.ts           # Your app
│   └── sw.js            # Service worker (generated by sw-builder)
├── public/
│   └── manifest.json    # Generated by manifest engine
└── package.json
```
